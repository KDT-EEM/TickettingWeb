# 프로젝트 주제
클라우드를 이용한 티켓팅 웹 서비스, 사용자가 뮤지컬, 콘서트, 스포츠 등의 다양한 이벤트 티켓을 구매할 수 있는 플랫폼 입니다.
사용자는 원하는 이벤트의 좌석을 선택하고, 로그인 후 구매할 수 있습니다. 마이페이지에서는 구매한 티켓을 확인 할 수 있습니다.

# 기술 스택
## 클라우드
NHN 클라우드
## DB
RDS(MySQL)
## 백엔드
PHP
## 협업 툴
git, github, NHN
